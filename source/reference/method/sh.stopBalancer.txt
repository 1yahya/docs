Definition
----------

.. method:: sh.stopBalancer(timeout, interval)

   Disables the balancer in a sharded cluster and waits for balancing to
   complete.


   .. list-table::
      :header-rows: 1
      :widths: 20 20 80
   
      * - Parameter
   
        - Type
   
        - Description
   
      * - ``timeout``
   
        - integer
   
        - Milliseconds to wait.
          
          
   
      * - ``interval``
   
        - integer
   
        - Milliseconds to sleep each cycle of waiting.
