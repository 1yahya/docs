Definition
----------

.. method:: db.changeUserPassword(username, password)

   Allows an administrator to update a user's password from within
   the shell.


   .. list-table::
      :header-rows: 1
      :widths: 20 20 80
   
      * - Parameter
   
        - Type
   
        - Description
   
      * - ``username``
   
        - string
   
        - Specifies an existing username with access privileges for this database.
          
          
   
      * - ``password``
   
        - string
   
        - Specifies the corresponding password.
          
          
   


   :throws exception:
         If an error occurs, the
         :method:`db.changeUserPassword()` helper throws an exception with the
         error message and code.
